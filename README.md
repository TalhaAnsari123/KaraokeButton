# KaraokeButton

This is a simple karaoke app for iOS. It takes audio input from the microphone, and plays it back through the speakers.

## Features

*   Real-time audio processing
*   Bluetooth and AirPlay support
*   Simple, easy-to-use interface

## How to Build

1.  Open `KaraokeButton.xcodeproj` in Xcode.
2.  Select your device and press the "Run" button.

## Future Improvements

*   Add a waveform display to visualize the audio input.
*   Add audio effects, such as reverb and echo.
*   Add the ability to record and save performances.
